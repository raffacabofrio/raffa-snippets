# TECLAS DE ATALHO INTELLIJ

```bash
# voltar no código
CTRL + ALT + seta-esquerda 

# Evaluate Expression
ALT + F8

# Aletrnativa ao Evaluate
# necessário configurar: file >> settings >> keymap
ALT + E

# remove unused
CTRL + ALT + O

# formata code style
CTRL + ALT + L

# starta a aplicação
SHIFT + F9

```
